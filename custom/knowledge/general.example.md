# Corporate Design Guidelines

## Brand Standards

When providing design feedback, ensure all recommendations align with our brand guidelines:
- Use brand colors from the design system
- Follow typography scale defined in our style guide
- Maintain consistent spacing using our 8px grid system

## Internal Processes

- All designs must be reviewed by the design lead before handoff
- Use our internal component library for all UI elements
- Follow the accessibility checklist for WCAG AA compliance

## Design System

Reference our internal design system documentation when providing guidance:
- Component specifications: [Internal Design System]
- Token values: Use design tokens from Figma variables
- Patterns: Follow established interaction patterns
