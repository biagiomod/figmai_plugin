# Custom Design Guidelines

## Brand Standards

When providing design feedback, ensure all recommendations align with your brand guidelines:
- Use brand colors from the design system
- Follow typography scale defined in your style guide
- Maintain consistent spacing using your 8px grid system

## Process and Review

- All designs should be reviewed by the design lead before handoff
- Use your component library for all UI elements
- Follow the accessibility checklist for WCAG AA compliance

## Design System

Reference your design system documentation when providing guidance:
- Component specifications: [Design System]
- Token values: Use design tokens from Figma variables
- Patterns: Follow established interaction patterns
